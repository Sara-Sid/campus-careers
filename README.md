# 🎓Campus Careers
A full-stack job board platform built for higher education — connecting job seekers, 
recruiters, and administrators at colleges and universities.

## 🌐Live Features
- **Browse Jobs** — Search, filter by category, location, and salary
- **Job Detail Pages** — View full job info and apply directly
- **Authentication** — Register and log in with Firebase Auth
- **Role-Based Dashboards** — Separate experiences for Seekers, Recruiters, and Admins
- **Admin Panel** — Multi-level admin system with user management, content management, and analytics

## 👤User Roles
 
| Role | Access |
|------|--------|
| **Job Seeker** | Browse jobs, save listings, track applications |
| **Recruiter** | Post jobs, manage listings, view candidate waitlist |
| **Admin** | Full control — manage users, delete posts and events  |
